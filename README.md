# Ethereum Foundation Website Clone

Ethereum Foundationの公式ウェブサイトをモダンなWeb技術で再現したクローンサイトです。

## 🌟 特徴

### デザイン
- **ダークテーマ**: 洗練された黒を基調としたデザイン
- **無限ガーデン**: メタファーを使ったコンセプチュアルなデザイン
- **モダンなタイポグラフィ**: Inter フォントを使用した読みやすいレイアウト
- **グラデーション効果**: 美しいカラーパレットとグラデーション

### インタラクティブ機能
- **スムーズスクロール**: セクション間の滑らかな移動
- **アニメーション**: スクロール連動のフェードイン効果
- **ホバーエフェクト**: カードやボタンのインタラクティブな反応
- **パーティクルシステム**: ヒーローセクションの動的な背景効果
- **タイピングエフェクト**: メインタイトルの段階的表示

### レスポンシブデザイン
- **モバイルファースト**: すべてのデバイスサイズに対応
- **ブレークポイント**: 1400px, 1200px, 992px, 768px, 576px
- **タッチフレンドリー**: モバイルデバイスでの操作に最適化
- **アクセシビリティ**: キーボードナビゲーションとスクリーンリーダー対応

## 🚀 セットアップ

### 1. ファイルの確認
すべてのファイルが正しく配置されていることを確認してください。

### 2. ローカルサーバーの起動
以下のいずれかの方法でローカルサーバーを起動してください：

#### Python 3を使用する場合
```bash
cd /Users/harukiiijima/Desktop/boderless/deck
python -m http.server 8000
```

#### Node.jsを使用する場合
```bash
cd /Users/harukiiijima/Desktop/boderless/deck
npx serve .
```

#### Live Server（VS Code拡張）を使用する場合
VS Codeで `index.html` を開き、Live Server拡張で起動

### 3. ブラウザでアクセス
- `http://localhost:8000` にアクセス
- または `http://127.0.0.1:8000`

## 🎨 カスタマイズ

### カラーパレットの変更
`styles.css` の `:root` セクションでカラーパレットを変更できます：

```css
:root {
    --primary-bg: #0a0a0a;
    --accent-color: #6262ff;
    /* その他のカラー変数 */
}
```

### アニメーションの調整
`script.js` でアニメーションの速度や効果を調整できます：

```javascript
// アニメーション速度の調整
const duration = 3 + Math.random() * 4; // 3-7秒のランダム
```

### コンテンツの編集
`index.html` でテキストコンテンツやセクションを編集できます。
