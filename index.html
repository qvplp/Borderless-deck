What is Ethereum?

Ethereum is a decentralized platform that enables smart contracts and decentralized applications (DApps) to be built and deployed without any downtime, fraud, control, or interference from a third party.

⚡

Smart Contracts

Self-executing contracts with terms directly written into code

Decentralized Apps

Applications that run on a peer-to-peer network of computers

🔒

Security

Cryptographically secure and tamper-proof by design

What is the EF?

The Ethereum Foundation (EF) is a non-profit organization dedicated to supporting Ethereum and related technologies. We fund research, development, and education to advance the ecosystem.
